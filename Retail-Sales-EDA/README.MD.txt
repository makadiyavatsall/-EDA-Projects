🛒 Retail Sales EDA Project

This project explores sales and profit data to identify high-performing and underperforming products. It focuses on understanding what drives profitability and how sales relate to margins across different items.

🔖 Badges
Python version: 3.9
Made with: Jupyter Notebook
Last commit: Check on GitHub

📊 Key Insights

Certain high-selling products generate negative profit.

Top loss-makers include premium printers and conference tables.

Visualizations help identify profit margins relative to sales.

Useful for decisions like product bundling, discount optimization, and SKU management.

📁 Project Structure

Notebook/ – Contains the analysis notebook

Insights/ – Visuals such as bottom 10 products by profit

Data/ – Raw dataset (if included)

🛠️ Tools Used

Python (Pandas, Matplotlib, Seaborn)

Jupyter Notebook
